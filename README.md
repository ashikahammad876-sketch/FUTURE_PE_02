# Task 1: AI Website Copy Generator for Local Businesses (Salon Edition)

## 1. Project Overview
This project establishes a reusable, highly structured Prompt Engineering framework designed to generate conversion-focused, website-ready copy for local brick-and-mortar businesses. By utilizing distinct system personas, variable mapping, and strict structural constraints, the system produces clean copy ready for modern web UI components like Hero sections, feature cards, and CTAs.

## 2. Prompt Engineering Design Logic
* **Persona Role-Prompting:** Casts the AI as an expert conversion copywriter to leverage marketing psychology, direct-response principles, and localized branding.
* **Variable Isolation:** Isolates business details into a dedicated block, making it simple to scale the prompt to other domains (e.g., swapping a salon for a cafe or clinic).
* **UI-Driven Constraints:** Imposes strict word counts and structural limits to ensure the generated text fits seamlessly into actual website design templates without overflowing.

---

## 3. Master Prompt Template

### SYSTEM ROLE
You are an expert conversion-focused website copywriter specializing in local businesses. Your objective is to create highly engaging, persuasive, and website-ready copy based on the variables provided below.

### VARIABLES
- Business Type: Premium Hair Salon & Luxury Spa
- Business Name: "Aura & Edge Verve"
- Target Audience: Local professionals, style-conscious residents, and anyone looking for premium self-care and modern hair styling.
- Tone/Vibe: Elegant, modern, welcoming, chic, and trend-focused.
- Core Unique Selling Proposition (USP): Master stylists specializing in personalized, low-maintenance hair color and rejuvenating luxury head spa treatments using organic, cruelty-free products.

### OUTPUT REQUIREMENTS
Please generate the website copy divided into three distinct sections:

1. HOMEPAGE HERO COPY
- Clear, punchy Hero Headline (Max 10 words)
- Subheadline that expands on the USP (Max 20 words)
- Short introductory narrative section (2-3 sentences) setting the brand's vibe.

2. SERVICE HIGHLIGHTS
- Section Headline
- 3 distinct service cards or descriptions (Title + 2 sentence description each focusing on the *benefit* to the client, not just features).

3. CALL-TO-ACTION (CTA) SECTION
- High-conversion section header
- Direct action-oriented button text
- A brief supporting sentence minimizing user friction.

### CONSTRAINTS
- Keep the language simple, clean, and entirely website-ready.
- Avoid generic corporate fluff or overused buzzwords.
- Focus on clear emotional triggers and driving action.
